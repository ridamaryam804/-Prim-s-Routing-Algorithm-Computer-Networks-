# -Prim-s-Routing-Algorithm-Computer-Networks-
# Project Structure
This folder contains the complete deliverables for our Computer Networks CEP project, which implements Prim’s Algorithm for network routing using Java Swing for GUI and visualization.

# Folder: `code/`
Contains the full source code and compiled `.jar` file:
CoreGraphHandler.java` – Handles graph data and logic.
Edge.java` – Defines the edge structure and weight.
PrimsRoutingSimulator.java` – Main class with GUI and routing simulation.
CN_CEP_2.jar` – Compiled runnable Java application.
# How to Run the Project
 Using the `.jar` File
1. Make sure Java 17+ is installed.
2. Open terminal or command prompt.
3. Navigate to the `code` folder.
4. Run the application with:

# bash
java -jar CN_CEP_2.jar
